Space-full-stack-website-
Developed a full-stack Python web application using MongoDB for data storage, focused on educating users about space. The homepage provides a dynamic feed of space facts and recent news updates. Users can read, save, and explore information, then test their understanding through interactive quizzes available anytime.

Flask-PyMongo==2.3.0
Flask-Login==0.6.2
