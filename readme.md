A full-stack Python Flask project for learning interesting space facts and taking quizzes.  
Designed for beginner-to-intermediate web development and database handling.

## Features

- Browse latest **space facts** directly on the homepage
- Take **quizzes** about planets, stars, and the solar system
- User authentication: **Register, Login, Logout**
- **Score tracking** for logged-in users (stored in MongoDB)
- Responsive design using **Bootstrap 5**
- Flash messages for feedback (success, warning, errors)

## Tech Stack

- **Backend:** Python, Flask, Flask-Login, Flask-PyMongo
- **Database:** MongoDB Atlas / Compass
- **Frontend:** HTML, Bootstrap 5, JavaScript
- **Environment management:** dotenv
